# Kanji App

Aplicación para practicar la escritura de kanjis japoneses.

## Estructura

- frontend → Flutter
- backend → Node.js

## Funcionalidades

- Dibujo de kanji
- Evaluación de trazos
- Sistema de puntuación

## Cómo ejecutar

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
flutter pub get
flutter run
